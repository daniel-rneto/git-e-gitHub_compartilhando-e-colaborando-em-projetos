# Git e GitHub: compartilhando e colaborando em projetos

### Comandos úteis do git:
- git add . (Adiciona todos todos os arquivos no repositório)

- git remote set-url origin git@github.com:daniel-rneto/git-e-gitHub_compartilhando-e-colaborando-em-projetos.git (trocar a chave ssh usada)

- ssh -T git@github.com (Testar conexão com o git)

- git remote -v (exibe uma lista de todos os repositórios remotos vinculados ao seu projeto)

- git remote set-url origin https://github.com/seu-usuario/seu-repositorio.git (Alterar a URL de um repositório remoto)

- git remote rename origin novo-origin (Alterar o apelido de um repositório remoto)

- git log (Mostra o histórico dos commits do repositório)
