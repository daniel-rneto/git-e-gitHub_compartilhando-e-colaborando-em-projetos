# Git e GitHub: compartilhando e colaborando em projetos

### Comandos úteis do git:
- git add . (Adiciona todos todos os arquivos no repositório)

- git remote set-url origin git@github.com:daniel-rneto/git-e-gitHub_compartilhando-e-colaborando-em-projetos.git (trocar a chave ssh usada)

- ssh -T git@github.com (Testar conexão com o git)
